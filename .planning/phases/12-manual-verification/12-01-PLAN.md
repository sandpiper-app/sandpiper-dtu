Create a test file with three `describe` blocks:

### Describe 1: HMAC End-to-End Verification

Create a `createCallbackServer` function matching the pattern from `packages/webhooks/test/webhook-queue.test.ts` lines 18-55 (copy the pattern exactly — it creates a local HTTP server on port 0 that records incoming requests).

Create a single test: `'HMAC e2e: sign -> deliver -> verify on receiver side'`

Steps:
1. Create an in-memory SQLite database and `SqliteDeadLetterStore` (same setup as existing webhook tests)
2. Create a `WebhookQueue` with `syncMode: true` and the DLQ
3. Create a `WebhookDelivery` with a known secret (`'test-webhook-secret'`), a payload (`{ id: 1, name: '#1001', total_price: '99.99' }`), topic `'orders/create'`, and the callback server URL
4. Enqueue the delivery
5. After delivery, get the received request from the callback server
6. **CRITICAL DISTINCTION**: Independently verify the HMAC using `crypto.createHmac('sha256', secret).update(receivedBody, 'utf8').digest('base64')` on the raw received body string. Do NOT use `generateHmacSignature` for verification — that would just test signing consistency, not the full sign-deliver-verify chain.
7. Assert the independently computed signature matches the `x-shopify-hmac-sha256` header on the received request
8. Also verify the received body parses to the original payload (body integrity check)
9. Clean up: `queue.shutdown()`, `close()` callback server, `db.close()`

Imports needed:

### Describe 2: Async Webhook Delivery Timing

Create a single test: `'async delivery: queued -> delivered within expected window'`

Steps:
1. Same setup (in-memory DB, DLQ, callback server)
2. Create a `WebhookQueue` with `syncMode: false` (async mode), `timeScale: 0.001` (compressed timing), and the DLQ
3. Record `enqueueTime = Date.now()`
4. Enqueue a delivery (first retry delay is 0, so delivery should be nearly immediate)
5. Poll for delivery using the Phase 9 polling pattern: loop every 10ms with a 2-second deadline, checking `requests.length > 0`
6. Record `deliveryTime = Date.now()` after request arrives
7. Assert `requests.length === 1`
8. Assert `elapsed = deliveryTime - enqueueTime` is less than 500ms (generous window for async overhead — first attempt has delay=0)
9. Clean up: `queue.shutdown()`, `close()`, `db.close()`
