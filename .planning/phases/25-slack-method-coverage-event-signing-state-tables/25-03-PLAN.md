<objective>
Fix three independent SLCK-16 bugs in the Slack twin's event and interaction delivery:

1. **EventDispatcher** currently sends `X-Shopify-Hmac-Sha256` (Shopify format). Fix to send
   `X-Slack-Signature` (`v0=<hex>`) and `X-Slack-Request-Timestamp` so Bolt's `verifySlackRequest`
   passes without error.

2. **InteractionHandler** generates a relative `response_url` (`/response-url/<id>`). Fix to
   generate an absolute URL using a `baseUrl` injected at construction time.

3. **Interaction routing** delivers to the event subscription URL. Fix to deliver to a separate
   interactivity URL registered via a new `POST /admin/set-interactivity-url` admin endpoint.

Purpose: SLCK-16 — Bolt's request verification contract requires Slack-format signatures and
absolute response URLs; mixing event and interaction endpoints breaks HTTPReceiver tests.

Output: event-dispatcher.ts (HMAC fix), interaction-handler.ts (baseUrl fix), admin.ts (new
endpoint), interactions.ts (routing fix).
</objective>

Replace the block that iterates `listEventSubscriptions()` with one that delivers to the
interactivity URL:
```typescript
// BEFORE (wrong — delivers to event subscription URL):
const subscriptions = fastify.slackStateManager.listEventSubscriptions();
for (const sub of subscriptions) {
  await fetch(sub.request_url, { ... });
}

// AFTER (correct — deliver to dedicated interactivity URL WITH Slack signature):
const interactivityUrl = fastify.slackStateManager.getInteractivityUrl();
if (interactivityUrl) {
  try {
    const formBody = `payload=${encodeURIComponent(JSON.stringify(payload))}`;
    // Sign the interaction payload with Slack HMAC (same format as events)
    const ts = Math.floor(Date.now() / 1000);
    const sig = `v0=${createHmac('sha256', signingSecret).update(`v0:${ts}:${formBody}`).digest('hex')}`;
    await fetch(interactivityUrl, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/x-www-form-urlencoded',
        'X-Slack-Signature': sig,
        'X-Slack-Request-Timestamp': String(ts),
      },
      body: formBody,
    });
  } catch (err) {
    request.log.warn({ error: err, url: interactivityUrl }, 'Failed to deliver interaction payload');
  }
}
```
