**Stage 2 (build):**
- WORKDIR /app
- Accept `ARG TWIN_NAME` (shopify or slack)
- Copy pnpm-lock.yaml, pnpm-workspace.yaml, package.json first (layer caching)
- Copy packages/ directory (all shared packages needed for build)
- Copy twins/${TWIN_NAME}/ directory
- Run `pnpm install --frozen-lockfile` with BuildKit cache mount for pnpm store: `--mount=type=cache,id=pnpm,target=/pnpm/store`
- Build shared packages first: `pnpm --filter="./packages/*" run build`
- Build the twin: `pnpm --filter="@dtu/twin-${TWIN_NAME}" run build`
- Deploy production dependencies: `pnpm deploy --filter="@dtu/twin-${TWIN_NAME}" --prod /prod/${TWIN_NAME}`

**Stage 3 (runtime):**
- Use `node:20-slim` base
- Accept `ARG TWIN_NAME`
- Set `NODE_ENV=production`
- WORKDIR /app
- Copy deployed production output: `COPY --from=build /prod/${TWIN_NAME} .`
- Copy compiled dist: `COPY --from=build /app/twins/${TWIN_NAME}/dist ./dist`
- Copy runtime file dependencies that tsc does NOT compile:
  - `twins/${TWIN_NAME}/src/views` -> `./src/views` (UI plugin resolves ../views from dist/plugins/)
  - `twins/${TWIN_NAME}/src/schema` -> `./src/schema` (GraphQL plugin's dist-mode path: ../../src/schema/)
  - `packages/ui/src/partials` -> `./node_modules/@dtu/ui/src/partials` (@dtu/ui getPartialsDir)
  - `packages/ui/src/public` -> `./node_modules/@dtu/ui/src/public` (@dtu/ui getPublicDir)
  Note: Use conditional COPY pattern — Slack twin has no src/schema dir, so copy only if exists. Since Docker COPY fails on missing source, use a glob pattern like `twins/${TWIN_NAME}/src/schema/schema.graphql*` or handle this by copying all of src/ selectively. The simplest robust approach: copy the entire `twins/${TWIN_NAME}/src/` directory to `./src/` — it only contains .ts files (ignored at runtime), .eta views, and .graphql schema, and adds negligible size.
- Copy healthcheck script: `COPY scripts/healthcheck.mjs ./healthcheck.mjs`
- Add `USER node` for security (non-root)
- EXPOSE 3000
- HEALTHCHECK using node healthcheck.mjs (interval=10s, timeout=3s, start_period=5s, retries=3)
