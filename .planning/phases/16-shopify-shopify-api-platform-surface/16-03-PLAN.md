    **Test 4 — begin returns 302 redirect:**
    The `begin` function uses the node adapter's `abstractConvertRequest` and `abstractConvertResponse`. It writes a 302 redirect to the rawResponse object. The node adapter writes the response via `res.setHeader` and `res.end` on a real ServerResponse. For testing, build mock objects that record what's written.

    Use a simpler approach — build a fake rawRequest and capture the redirect via a mock rawResponse:
    ```typescript
    it('begin returns a redirect to the Shopify OAuth authorize URL', async () => {
      // build minimal IncomingMessage-like mock for begin()
      const mockReq = {
        method: 'GET',
        url: 'https://test-app.example.com/auth?shop=dev.myshopify.com',
        headers: { host: 'test-app.example.com' },
      };

      // build a mock ServerResponse that captures headers
      const capturedHeaders: Record<string, string> = {};
      let capturedStatus = 0;
      const mockRes = {
        setHeader: (name: string, value: string) => { capturedHeaders[name.toLowerCase()] = value; },
        writeHead: (status: number) => { capturedStatus = status; },
        end: () => {},
      };

      // begin() returns an AdapterResponse — node adapter also writes to rawResponse
      const response = await shopify.auth.begin({
        shop: 'dev.myshopify.com',
        callbackPath: '/auth/callback',
        isOnline: false,
        rawRequest: mockReq as any,
        rawResponse: mockRes as any,
      });

      // The Location header must point to the Shopify OAuth authorize URL
      // Check either the returned AdapterResponse headers or the mock rawResponse headers
      const location = (response as any)?.headers?.get?.('location')
        ?? capturedHeaders['location'];
      expect(location).toMatch(/\/admin\/oauth\/authorize/);
      expect(location).toMatch(/client_id=test-api-key/);
    });
    ```

    PITFALL: The node adapter's `abstractConvertResponse` expects a real Node.js `ServerResponse`. The mock above only captures the setHeader call.
